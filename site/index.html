OpenMarket

A public good marketplace contracts with full onchain data to trade NFTs without platform fees nor API key

View on Github

Mission

The aim of this project is to propose an alternative solution for NFT creators in the zero-royalties era. This project showcases public good marketplace contracts with full on-chain data to trade NFTs without any platform fees or the need for an API key.

NFT Royalties Are Dead

The major marketplaces such as Blur, OpenSea, and Magic Eden have recently stopped enforcing NFT royalties as standard. This zero-royalty battle has killed the model that allowed creators to release NFTs for free or at a low cost and earn royalties.

‚õî

Off-Chain Order Books

Leading NFT platforms have implemented off-chain order books that are not publicly accessible. To integrate with these marketplaces, projects have to request API keys, like in web 2.0, and the approval process can take several months.

üöß

Limited Distribution

There's a lack of NFT distribution channels. The existing NFT marketplaces have positioned themselves as "the distributors" of NFT collections. They decide on the rules, service fees, access to data and so on.

Demo

This demo uses images forked from the NounsDAO NFT collection.

OpenMarket smart contracts are deployed on Linea Goerli testnet. The Nouns OpenMarket smart contract address on Linea Goerli: 0xfB27451439336cf06B719a6117BF492D245a920a

This demo represents two trading sides:

  1. A seller, who wants to sell their NFT
  2. A buyer, who can purchase any NFT from a single collection that is currently on sale.

Sell NFT

As an NFT owner, you can put on sale the NFT that you possess from a given collection. In case if you possess multiple NFTs you can choose which you'd like to sell.

Buy an NFT

As a buyer, you can view the NFT asset(s) of a particular collection that are available on sale, and buy an NFT of your choice.

Roadmap

OpenMarket is already more gas efficient than common marketplaces due to its simplicity for buying assets: Less than 100k gas!

  1. We plan to add the ability to register orders on L2 chains, to buy items for collections on mainnet, reducing the listing fees in the future.
  2. We also plan to add a way to retrieve collection metadata from IPFS, to enable users search by traits, etc.
