Mission

The aim of this project is to suggest an alternative solution to NFT creators in zero-royalties era. This project showcases public good marketplace contracts with full onchain data to trade NFTs without platform fees nor API key.

NFT Royalties Are Dead

The major marketplaces such as Blur, OpenSea, and Magic Eden have recently stopped enforcing NFT royalties as standard. This zero-royalty battle has killed the model that allowed creators to release NFTs for free or at a low cost and earn royalties.

‚õî

Off-Chain Order Books

Leading NFT platforms have implemented off-chain order books that are not publicly accessible. To integrate with these marketplaces, projects have to request API keys, like in web 2.0, and the approval process can take several months.

üöß

Limited Distribution

The existing NFT marketplaces have positioned themselves as "the distributors" of NFT collections. They decide on the rules, service fees, access to data and so on.

Demo

This demo represents two trading sides:

  1. a seller, who wants to sell his NFT and
  2. a buyer, who can buy any NFT from a single collection that are currently on sale.

This demo uses images forked from the Nouns DAO NFT collection.

Sell NFT

As an NFT owner, you can put on sale the NFT that you possess from a given collection. In case if you possess multiple NFTs you can choose which you'd like to sell.

Buy an NFT

As a buyer, you can view the NFT asset(s) that are available on sale and buy a chosen one from the interface below.
